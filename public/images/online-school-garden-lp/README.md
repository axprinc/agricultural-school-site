このフォルダは、オンラインスクール用LP（`/online-school-garden-lp.html`）で使用する画像を置くためのディレクトリです。

- 例: `hero.jpg`, `step-1.jpg`, `benefit-vegetables.jpg` など
- HTML からの参照例:
  - `<img src="/images/online-school-garden-lp/hero.jpg" alt="ヒーロー画像">`

このフォルダに画像ファイルを追加すると、ブラウザからは次のようなURLで参照できます:

- `http://localhost:3000/images/online-school-garden-lp/ファイル名`
